# Вики портала Shectory

Документация для инфраструктуры и общих соглашений экосистемы **Shectory** (портал, прикладные проекты на VDS, деплой).

**Расположение в git:** каталог `docs/wiki/` в репозитории **ourdiary** — общий ориентир для всех прикладных проектов (существующих и новых). При клоне только других репозиториев откройте этот файл в клоне `ourdiary` или скопируйте раздел в свой проект.
