# Наш дневник

Семейная социальная сеть с уклоном в календарь, учёт расходов, рейтинг участников и планирование событий.

## Описание

**Наш дневник** — веб-приложение для семьи, где каждый член семьи может:

- 📖 **Вести дневник** — записывать события, воспоминания, важные моменты
- 📅 **Планировать** — культурные мероприятия, поездки, праздники на 1–2 месяца вперёд
- 💰 **Учитывать расходы** — совместный семейный бюджет по категориям
- ✅ **Выполнять задачи** — домашние дела, поручения, трекинг обязанностей
- ⭐ **Зарабатывать очки** — рейтинг «семьянина» за активность
- 📺 **TV-режим** — красивый дашборд для большого экрана 75" в гостиной

## Технологический стек

- **Frontend/Backend**: Next.js 16.2.2 (App Router), React 19, TypeScript
- **Стили**: Tailwind CSS v4
- **База данных**: PostgreSQL + Prisma 7
- **Аутентификация**: NextAuth v4 (CredentialsProvider + JWT)
- **Иконки**: lucide-react
- **Утилиты**: date-fns, clsx, tailwind-merge

## Быстрый старт

```bash
# Установить зависимости
npm install

# Настроить переменные окружения
cp .env.example .env
# Отредактировать .env: DATABASE_URL, NEXTAUTH_SECRET, NEXTAUTH_URL

# Применить миграции
npx prisma migrate deploy

# Создать суперадмина
npx prisma db seed

# Запустить в режиме разработки
npm run dev
```

Приложение доступно на http://localhost:3000
